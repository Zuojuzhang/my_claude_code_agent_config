# 课程营销定位方法

本文件用于个人 IP、课程和知识产品任务，来源于本地《课程营销学AI版书稿》的定位篇，并与通用定位理论对齐。课程方法属于项目方法论，不能替代外部市场证据。

## 一、课程定位总结构

课程方法使用四层定位：使命、人设、商业、内容。执行顺序由内到外：

1. **使命定位**：为什么做，为谁创造什么差异价值。
2. **人设定位**：为什么是你，哪些专业和人格资产能建立信任。
3. **商业定位**：在哪个细分市场、以何种产品价格渠道组合参与竞争。
4. **内容定位**：具体解决哪些用户任务、痛点和行为改变。

四层必须互相一致。使命、人设与产品能力冲突时，不能用表达补救。

## 二、课程定位公式

课程营销方法中的压缩公式：

> 定位 = 差异化标签 + 品类

使用限制：

- 品类先帮助用户理解“这是什么”。
- 差异化标签再回答“为什么优先选它”。
- 标签必须相对于竞品成立，并与用户重要需求相关。
- “第一”“唯一”只有在边界、证据和可持续性明确时才能使用。

## 三、使命定位

方法公式：

> 使命定位 = 场景角色 + 利他目标 + 差异化标签

分析问题：

- 在什么场景中，定位对象扮演什么可信角色？
- 希望帮助哪类人达成什么结果？
- 该价值与大量同类使命表达相比有何具体不同？
- 使命是否能约束不该做的业务，而不只是美好口号？

报告字段：`subject.identity`、`subject.objective`、`executive_summary.core_value`。

## 四、人设定位

人设不是表演，分析三个层面：

1. **专业人设**：经历、方法、结果、资质、作品和行业组合形成的专业差异。
2. **性格人设**：稳定、真实且与交付方式一致的品格优势。
3. **用户人格匹配**：目标用户愿意信任、学习和持续互动的沟通方式。

判断规则：

- 个人经历只能作为自述，需案例、作品、客户或第三方证明后才能升级为事实。
- 人格特质不能代替专业能力。
- 热门内容形象与售卖产品不一致时，信任迁移成本上升。

## 五、商业定位

先缩小可研究的市场边界，再分析竞争：

- 地域、语言、渠道、客群、场景、交付方式和价格区间必须明确。
- 默认使用产品、价格、渠道、传播四个维度建立蓝海差异化表。
- 个人或小团队优先选择能真实交付且资源匹配的位置，不与机构全维度竞争。

三类机会路径：

1. 找到现有市场的重要空白。
2. 重新定义市场或建立可理解的新类别。
3. 为过度竞争市场提供基础设施、方法或服务。

空白只有在“需求重要 + 现有供给不足 + 自身能交付”同时成立时才是机会。

## 六、内容定位

内容定位不从“我会什么”开始，而从用户任务路径开始：

1. 列出用户从触发到结果的行动路径。
2. 在每一步识别知识、能力、情绪、时间、资源和环境阻碍。
3. 使用具体的“靶心人”帮助验证抽象用户画像。
4. 标注竞品重点解决和未充分解决的痛点。
5. 选择一个重要、紧迫、现状不满且可交付的核心痛点。
6. 把课程设计为能产生结果的“止痛药”，而不是知识目录。

## 七、课程专用报告扩展

个人 IP 或课程报告必须额外包含：

- 使命、人设、商业、内容四层一致性矩阵。
- 专业人设、性格人设与用户人格匹配证据。
- 靶心人、行动路径和痛点定位表。
- 课程结果、方法机制、交付结构和内容差异。
- 标题、内容、产品、价格、渠道和销售表达的定位映射。

结构化报告使用 `course_marketing_extension`，强制记录：

- 专业人设、性格人设和用户人格匹配。
- 一个具体靶心人，以及从触发到结果的行动路径、痛点和当前方案。
- 产品、价格、渠道、传播四个维度的市场基线、差异选择和证据。
- 课程结果、方法机制、交付结构和内容差异。

4P 空位仍须通过“需求重要、供给不足、自身可交付”三项机会门，不能因为表格出现空白就直接命名为蓝海。

## 八、否决规则

- 使命宏大但没有特定用户、场景和差异：否决。
- 人设只靠自我描述，没有作品或结果：降级为待验证。
- 市场空白只因未找到竞品：否决。
- 痛点重要但课程无法交付结果：否决。
- 内容独特但用户不愿为结果投入时间或金钱：不进入核心定位。
- 四层中任一层与推荐定位直接冲突：先修正方案，不直接输出执行建议。
