# 去AI腔 no-ai-style

规则正文唯一权威是 `~/.claude/rules/no_ai_style.md`。这份skill不另存规则，避免两处定义drift。

被触发时（生成可读文本 / 用户丢一段文本要去AI味 / 讨论某段文字AI味重不重），按以下流程：

1. Read `~/.claude/rules/no_ai_style.md`，拿到全部规则与个人书写偏好。
2. **自己生成文本**：输出前按规则逐条自检，命中即改。
3. **重写他人文本**：通读标记所有命中位置，逐条重写保留原意，整句语感一起改，不只删触发词。输出重写版，如用户需要再用3句内说明主要改了什么。
4. **分析AI味**：精确到句子和触发词引用原文，不空泛说「整体AI味重」。

最终检验：把输出读出来，像普通同事说话就过关；像演讲稿、发布会、公众号、培训PPT就重写。
