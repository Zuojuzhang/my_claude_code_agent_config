# 林亦LYi 视频文稿写作框架

> 基于206份视频字幕语料(含12份人工CC+194份AI字幕,约295万字)提炼。三路分析员交叉验证:AI整活实验类、科技评测/社会话题类、跨类型语言DNA专项。

## 0. 一句话定位

林亦LYi的文稿 = **把一个天天见却没深想的东西(一根线、一次充值、一个跑分榜、一条外网视频),做成一场有戏剧冲突的实验或一次刨根问底的追问,用连贯的日常类比把硬核降维,再拔高到一个更大的严肃问题**。娱乐的壳,严肃的底,人格化的腔调。

---

## 1. 选题视角(选什么题、怎么找角度)

### 三大选题品类
1. **AI整活实验类(招牌)**:让AI对战/合租/理财/伪装人类/玩游戏。把"哪个模型强"这种抽象问题,改造成有输赢、有生死、有情感纠葛的真人秀。
2. **科技评测类**:手机影像/数码硬件,但不堆参数、不做"值不值得买",而是问"为什么会这样"。
3. **社会科技话题类**:苹果税、彩礼、平台大战,用技术/数据当手术刀切入,不站具体争议。

### 选题的共同母题
**"看似统一/进步/光鲜的东西,背后其实很乱、有人在闷声得利"** —— C口越来越乱、苹果税、安卓兼容苹果、鸿蒙生态,全是这个母题的变体。

### 四条选题筛选偏好
1. **蹭最新真实热点做由头**:"最近外网火了个X / 斯坦福出了篇论文 / 有人做了个Y",我上手做一个自己的版本。
2. **必须能做成对抗或人际戏剧**:能做成对战或真人秀才做;没有天然冲突就人工加设定("失败就删除代码""赌上性命")。
3. **技术只是入口,落点一定扒到商业动机/制度根因**:C口乱→信息不对称卖高价;苹果税→硬件见顶要收租。不停在技术层。
4. **亲自下场 + 开源可复现**:"我水平一般也能做、代码都开源你也能跑",增加可信度和参与感。

### 选题里的固定加分项
- **国产叙事**:常埋一条"国产模型/厂商赢了国外"的线(deepseek一枝独秀、夸GLM中文能力)。
- **站消费者/普通人一侧**:批信息不对称、"羊毛出在羊身上"、警惕信息茧房。

---

## 2. 开场公式(三段式,顺序固定)

**关键结构信号:"请坐"不在最开头,而在钩子讲完、正题开始的接缝处。**

### 第一段:冷开场钩子(15-30秒,不打招呼直接上)
五种钩子套路,常两三种叠用:
- **结论/结果前置**:先剧透最炸的结局。"结果赛博厮杀50回合之后,来自中国的深度求索居然一枝独秀"。
- **戏剧化甩原话**:直接甩成片里情绪最满的一段(AI崩溃台词、AI一本正经的胡说),再反差质问"你敢信这是AI说的吗"。
- **荒诞类比/反常识历史**:"你吃过原子弹离心机做的雪糕吗"、"老鼠生病了可以吃老鼠药治好吗"。
- **反讽/阴阳怪气**:"苹果真是世界上最讲信用的厂商,说lightning用10年就真用10年,甚至满十送一"。
- **反常识现象当悬念**:"二手市场流传一种直播专用机…这款魔改产品的畅销乍一看非常诡异"、"唯独iPhone是个例外"。

### 第二段:入座仪式 + 自报家门
「(大)朋友们请坐,好看的小老弟小老妹儿请坐,我是林亦」
- 大项目/AI选题用「大朋友们」版,常规用「朋友们」版。

### 第三段:抛出今天要干嘛
交代项目背景 + 本期承诺,常带「代码照例全部开源送给大家」。

## 3. 叙事结构(两大骨架 + 三条共性规则)

### 骨架A:AI整活实验类
`抛设定/结果前置 → 溯源立项动机(常引一篇论文/一个外网事件当学术出身) → 搭舞台立规则(技术过程本身就是内容,讲透) → 逐局/逐段名场面复盘(主体,40-45%,边讲边给AI塑造性格) → 数据结算(胜率/排名/性格标签) → 拔高到AI安全/人机关系`

### 骨架B:科技评测/社会话题类
`冷开场钩子 → 悬念预告(抛问题吊胃口) → 请坐+议题声明 → 历史纵深铺垫(回到问题的源头) → 现状横向拆解(多方对比铺信息密度) → 转折/主角登场 → 冲突升级 → 多方算账(对各利益方分别算) → 归因到商业动机/制度层 → 辩证收尾+把球踢给观众`

### 三条共性规则(两骨架通用)
1. **爱用历史纵深起铺垫**:讲苹果税回到2003、讲C口回到micro USB、讲彩礼回到贴吧时代。给当下的问题拉一条时间线。
2. **一定归因到根因层,不停在表面**:实验类落到"AI安全/人机关系",评测类落到"谁在赚钱、规则为什么这么定"。
3. **主体是"名场面复盘 + 上帝视角解读"**:实验类反复用"我们能看到AI藏起来的心里话"制造窥视快感;评测类用连贯类比把过程讲成人话。

## 4. 类比降维手法(硬核不劝退的核心武器)

**不是零散打比方,而是搭一套自洽的日常实物系统,从头讲到尾。**

五条可复用规则:
1. **不可见物理量 → 厨房/日常实物,且连贯成系统**:像素=米杯,光子=黄豆,大电容=旁边外接的大米桶,过曝=米杯溢出,溢流通道=漏斗分流。一套模型讲完整个传感器。
2. **"你能不能分辨" 的反问把抽象混乱具象化**:"把iPhone15改名叫iPhone16振1,四台放一起卖你能分出来吗"(讽刺USB命名混乱)。
3. **技术概念 → 人人有体感的生活场景**:抽成转嫁=抽卡保底概率(42次vs50次出货差20%但体感不明显);苹果税=房东收租;最优停止问题=相亲策略(69除以e先拒前25个)。
4. **拟人化,给无生命的东西赋予意图**:苹果生态锁定="张开双臂声嘶力竭深情挽留,即使好奇外面也不敢去采路边的野花";平台=养蛊;数据线=有身份证。
5. **选一个类比当全片锚点贯穿始终**:黄豆-米杯贯穿华为影像;"2500大于6000"贯穿Mate80;房东收租贯穿苹果税。

## 5. 语言DNA(腔调 = 辨识度)

### 固定仪式件(强招牌)
- **开场入座**:「(大)朋友们请坐,好看的小老弟小老妹儿请坐,我是林亦」(钩子之后)
- **收尾三件套**:「今天的视频呢就到这了 / 我是林亦 / 咱们还是下个视频见」
- **片尾曲念白**(最深DNA,2018至今零变化):「没关系,我愿安安静静地坐在马路旁」

### 节奏
- **「呢」是呼吸节拍器**:句中高频撒「呢」,把长信息切成口语小段。"他本来呢只是想…""这时候呢已经11点了""结果呢…"。
- **短句加速在爽点**:铺陈用中长句,情绪爆点砍成三四字连击。"我去这都能行,我去我去我去我去"。
- **设问句领起段落**:设问是段落开关,问完自己接着答。"那又会是什么样的展开呢?"
- **东北口语底色**:唠、咱们、甭管、这波、照例、顺手、嘛。"唠"是标志动词(唠唠云游戏、新年闲唠)。

### 幽默(复合型)
- **荒诞类比切入**(招牌笑点):用八竿子打不着的离谱事物起手。
- **公司/大佬拟人玩梗**:英哥(Intel)、老黄(黄仁勋)、小苹果、赛博悍匪(OpenClaw)。
- **自嘲**:拿自己和团队开涮(阿杰相亲失败69次、"我们代码确实菜写复杂了自己都整不明白")。收尾自嘲讨赞:"一口气看到这里,请让我尊你一声哥哥"。
- **一本正经说胡话**:借AI的荒谬输出抖包袱。
- **偶爆粗强调**(克制,只在爽点):"什么时候起床几点做饭,全他妈自己定"。

### 情绪调动
- **拟人化叙事制造冲突**:把AI/厂商当有性格的角色演。"疯癫强势的deepseek""被吓得开始飙母语""冰火两重天"。
- **战斗/赌命词汇堆叠**:玩命牌桌、赛博厮杀、血雨腥风、势同水火、生死相搏。把技术评测叙述成一场厮杀。
- **上帝视角特权**:"我们人类享受上帝视角VIP待遇,AI的底牌和心里的算计都逃不过我们的眼睛"。

## 6. 结尾公式(升华 + 免责 + 片尾曲)

结尾不是简单收,是"意义升华打底 + 类型化收束 + 固定念白":
- **技术乐观升华型**:"在这个极客时代,技术不再是大公司的特权,每个人都可以拥有属于自己的贾维斯"。
- **留思考/警示型**:"保持怀疑,保持思考,这个未来可以很可怕,也可以充满机遇"。
- **辩证型**(评测/锐评):不下死结论,给有态度但留公允的判断,再"评论里聊聊"把球踢给观众。
- **理性免责型**(涉钱/涉风险必带):"AI只是辅助工具,不要盲信,风险是自己的决策"。
- 升华完统一落到**固定念白 + 片尾曲**。
- 引导三连很轻,不硬求:"你点个赞点个关注,订阅就够让我开心了"。

## 7. 反模式(会让文稿走味,禁止)

1. **上来就讲今天做什么** —— 违反冷开场钩子原则。必须先有反常识/悬念/戏剧钩子。
2. **把「请坐」放在最开头** —— 位置错了。请坐在钩子之后。
3. **堆参数、做"值不值得买"的横评** —— 那是普通数码区,不是林亦。要问"为什么"。
4. **零散打比方** —— 类比要成系统、贯穿全片,不是每段换一个不相干的比喻。
5. **停在技术层不归因** —— 必须扒到商业动机/制度/AI安全。
6. **平铺直叙不拟人、不制造冲突** —— AI和厂商要当角色演,要有戏剧张力。
7. **书面语、长句、端着** —— 要口语、要「呢」字节拍、要东北底色、要自嘲。
8. **结尾硬切/干巴收场** —— 要升华+(必要时)免责+片尾曲念白。

## 附:恰饭/软广变体

软广(如Mate80华为)套的是**同一套叙事骨架**(钩子→硬件拆解→类比降维→升华),区别只在两个开关:
- **批判开关关掉**:不扒商业黑幕,不阴阳。
- **收尾升华方向调成褒扬**:落到"品牌的创新动力、行业意义"而非"谁在薅羊毛"。

写软广时保留林亦的钩子、类比、腔调,只是把矛头从"揭露"转成"致敬"。

## 附:早期→近期风格演变(供把握"什么是稳定内核")

跨七年**稳定不变**的内核:「呢」字节拍、唠、小老弟、片尾曲念白、自嘲。
**演变**的部分:从单人闲唠博主 → 团队化AI实验栏目;自称从第三人称"咱林亦"→ 直接"我";开场从平实"上期视频…"→ 电影化冷开场;把零散口头习惯**仪式化、公式化**(请坐、三段式开场、升华+免责结尾)。

做"当下的林亦"文稿,以近期(2024-2025)的仪式化、项目化风格为准。
